# Multi-Tenant Feature Flag Management System

## Tech Stack

- React + Vite
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Bootstrap

## Features

### Super Admin
- Login
- Create Organization
- View Organizations
- Update Organization
- Delete Organization

### Organization Admin
- Signup
- Login
- Create Feature
- View Features
- Update Feature
- Delete Feature

### End User
- Check Feature Status

## Installation

### Backend

cd backend
npm install
npm start

### Frontend

cd frontend
npm install
npm run dev
